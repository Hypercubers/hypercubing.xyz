# Physical 1×1×1×1

!!! info inline end "Physical 1×1×1×1"
    <!-- revert to rubber duck if picsum stops working -->
    <!-- ![Rubber duck](https://assets.hypercubing.xyz/img/phys/rubber_duck.jpg) -->
    ![Random photo](https://picsum.photos/1000)

    **4D Shape:** Hypercube

    **Physical Shape:** Literally anything

    **Pieces:** 1

    **Magnets:** 0 (typically)

    **Completed:** ~3,000,000 BCE[^earliest-tool]

1^4^ has one state. Any physical object can represent that.

If you require a physical puzzle to include an object for each sticker, then any collection of 8 physical objects can be a physical 1^4^.

If you consider the state of a puzzle to include its orientation, then a 1^4^ has 192 states because a hypercube has 192 orientations. Typically, however, we only require physical puzzles to represent all possible twists, not all possible full-puzzle rotations. For example, the [Tymon's 1×3×3×3](1x3x3x3.md) has no way to reflect the core, so some 4D symmetries cannot be represented using the physical puzzle.
